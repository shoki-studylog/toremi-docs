Privacy Policy

Last Updated: January 1, 2025

Introduction

This Privacy Policy describes how Trend Addict Toremi ("the Operator") handles information in connection with the personal web application used for managing TikTok and YouTube content. This application is for the Operator's personal use only and does not collect data from any third-party users.

Information Accessed

The application may access the following types of information through TikTok and YouTube APIs:

· Account information (channel name, profile data)
· Content metadata (titles, descriptions, upload dates, tags)
· Performance analytics (views, likes, comments, engagement)
· Access tokens required for API authentication

How Information Is Used

All accessed information is used exclusively by the Operator for the following personal purposes:

· Managing and scheduling content across platforms
· Reviewing channel performance and analytics
· Organizing and archiving content data
· Improving personal content creation workflows

No Third-Party Sharing

The Operator does not sell, share, rent, or disclose any accessed data to third parties. All data remains private and is used solely by the Operator for personal content management.

Third-Party APIs

This application integrates with the TikTok API and YouTube Data API. Data retrieved through these APIs is subject to their respective privacy policies. The Operator complies with all applicable platform developer agreements.

Data Storage & Security

Any stored data (such as access tokens or cached content metadata) is kept locally or in a private environment accessible only to the Operator. No data is stored on public servers or exposed to outside parties.

Changes to This Policy

This Privacy Policy may be updated at any time. The most current version will always be available on this page.

Contact

For any questions regarding this Privacy Policy, please contact the Operator through the official social media channels.
